# NetcoreMessenger-ANDROID 💬

Добро пожаловать в репозиторий проекта **NetcoreMessenger**! Это современный, полнофункциональный мессенджер с видеозвонками, кружочками и профилями, созданный с нуля.

*(Весь код бэкенда и Android-клиента объединен в этом репозитории для удобства презентации).*

## 🌟 Концепция проекта

Мы привыкли к Telegram, WhatsApp и другим мессенджерам.**NetcoreMessenger** создавался как проект для себя похожий на привычные нам мессенджеры для полного удобства.


## ✨ Полный список возможностей

### 📞 Связь и Звонки
- **Аудио-звонки (P2P):** Прямое соединение между пользователями без задержек.
- **Видео-звонки (P2P):** Высокое качество видео через протокол WebRTC.
- **Уведомления о звонках:** Даже если приложение закрыто, пользователь получает стилизованное пуш-уведомление "Входящий звонок" с возможностью поднять или сбросить трубку.

### 🎥 Медиа-сообщения
- **Видеокружки (как в Telegram):** Знаменитый формат мгновенных видеосообщений в круглом интерфейсе.
- **Голосовые сообщения:** Классический аудиоформат для быстрых ответов.
- **Фото и Альбомы:** Отправка одиночных изображений или целых галерей с красивой сеткой отображения в чате (Grid Layout).

### 💬 Текстовые чаты
- **Real-time обмен:** Сообщения доставляются мгновенно благодаря WebSockets.
- **Статусы "В сети" и "Печатает...":** Полный эффект присутствия собеседника.
- **Управление сообщениями:**
  - Удаление сообщений (у всех).
  - Редактирование.
  - Пересылка в другие чаты.
  - Ответы с привязкой к оригинальному сообщению.

### 👤 Социальные функции
- **Профили пользователей:** Аватарки, описание "О себе", имена, юзернеймы.
- **Юзернеймы (@username):** Глобальный поиск по всей базе пользователей без необходимости знать номер телефона.
- **Просмотр профиля:** Возможность в один клик открыть профиль собеседника и посмотреть его данные.

## 🛠 Техническая изнанка (Простым языком)

Чтобы все это работало, используется связка современных технологий:

1. **Android Клиент (Kotlin + Jetpack Compose):**
2. **Сигнальный сервер (WebRTC):** Чтобы два телефона могли соединиться по видео, им нужно "найти" друг друга в интернете. Сервер мгновенно обменивается координатами телефонов, после чего они связываются напрямую (P2P).
3. **Бэкенд (Python + FastAPI):** Сервер, который обрабатывает авторизацию, хранит историю переписок и раздает картинки.
4. **Защита и безопасность (JWT + RSA):** Все сессии пользователей зашифрованы криптографическими ключами.
